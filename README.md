# necoo-loader
## Introduction
> A webpack loader help you to learn Javascript source code. Use code to learn code.

## Usages

`npm i` or `cnpm i`

`npm run dev`


## Examples

```js
import {observable} from "mobx";
window.necooIndex = 0;
let t = {
    d: {
        e: {
            f: 2
        }
    }
};
let b = observable(t);
```
